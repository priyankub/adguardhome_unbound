# Security Policy

## Supported Versions

We actively maintain and patch the container image built from this repository. Because this project follows a continuous delivery GitOps workflow, only the latest release tag receives active security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

**Please do not open a public GitHub Issue for security vulnerabilities.**

If you discover a security flaw or an issue within this container configuration, please report it through one of the following channels:

1. **GitHub Private Vulnerability Reporting:** Navigate to the "Security" tab of this repository, click on "Advisories", and select "Report a vulnerability".
2. **Direct Contact:** Email the maintainer privately at `priyanku@baruah.net`.

Once received, we will acknowledge your report within 48 hours and work on providing an automated patch update via our Renovate and GitHub Actions automation loops.
